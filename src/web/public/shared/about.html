About

classic-vibe-mac is a GitHub template that lets you write a classic Mac OS application in C, push it to a repo, and have it running in the browser on every commit, with zero local emulator setup.

The pipeline:

  • You write C against the Mac Toolbox in src/app/.
  • GitHub Actions cross-compiles it with Retro68 for the 68k.
  • The compiled app is baked into a bootable System 7.5.5 disk image's Startup Items folder.
  • A stripped-down BasiliskII WebAssembly core boots that disk on GitHub Pages.

This Reader app is the on-brand demonstration: instead of building yet another Minesweeper clone, the template ships a small native Mac app that consumes content the host page provides over a virtual shared volume.
