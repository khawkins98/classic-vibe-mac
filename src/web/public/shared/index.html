Welcome to Reader

You are looking at an HTML viewer written in C, running inside a 68k classic Mac OS emulator, in your browser. The page you are reading was fetched from a folder on the host page that is mounted as a Mac volume named Shared.

This is a tiny demo of what the classic-vibe-mac template can do: write a native Mac app in C, ship the source through GitHub Actions, have it running on the open web with no installation.

Tips

Click an underlined link to follow it. Press Backspace (or pick View > Back) to return. Cmd-R reloads. Cmd-O opens the standard file dialog so you can pick any HTML file from the shared volume.
