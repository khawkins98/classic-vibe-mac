Spike: WASM-Rez (mini), in-browser resource compiler

This page hosts the mini variant of Retro68's Rez compiled to WebAssembly. The "mini" variant ditches Boost.Wave (the upstream C preprocessor) and substitutes a hand-rolled lexer; it covers the slice of Rez needed for resource-fork-only edits with the type definitions inlined into the .r file (or, in a follow-up, emitted by a #include-aware stage).
